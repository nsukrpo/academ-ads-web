# Web-application "Academ ads" for admin

This application was developed for subject "Collective software development".

Academ ads - ad placement for students in Akademgorodok, Novosibirsk.

### A user can:
- create ads
- book ads
- buy good or service in ads
- chat with a seller
- add an ad to favourites
- and search necessary ads
